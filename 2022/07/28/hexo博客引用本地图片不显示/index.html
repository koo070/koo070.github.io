hexo博客引用本地图片不显示

发表于 2022-07-28 分类于 hexo

主要原因时是post_asset_folder插件的版本不对

方案步骤

1. 修改配置文件

修改文件_config.yml 里的post_asset_folder:这个选项设置为true
这个时候创建新的文章时会在_posts文件下自动生成一个与新建文件同名的文件夹。

2. 准备图片

将要引用的图片放到文件夹hexo博客引用本地图片不显示下。(假设图片名为“01”)

3. 安装插件hexo-asset-image

使用以下命令：

npm install https://github.com/CodeFalling/hexo-asset-image –save

注意 不要使用npm install hexo-asset-image --save这个命令安装,版本号不对

4. 在md中书写格式（相对路径）

在md使用如下形式引用图片

![01](hexo博客引用本地图片不显示/01.jpg)

5. md文件名称和图片存放文件夹名称不同时

注意 重命名md文件之后记得重命名文件夹，当md文件的名称与文件夹名称不同时，映射关系以md文件名称为主。会导致src图片路径找不到

总结

  1. 检查_config.yml 文件的post_asset_folder:选项是否为true
  2. 将要引用的图片放到文件夹kk下
  3. 安装插件hexo-asset-image
  4. 在md中引用图片
  5. 检查md文件的文件名称和图片存放文件夹名称是否相同
